VANASTREE WEBSITE — IMAGE FOLDER
=================================

Drop your original photos into this folder using these exact filenames,
and they'll automatically appear in the right spots on the site:

  hero-seeds.jpg        -> Top banner on every page (recommended 1180x290px)
  women-of-forest.jpg   -> Home page, "Women of the Forest" section (360x220px)
  about-founder.jpg      -> About page photo (360x220px)
  education-camp.jpg     -> Education page photo (360x220px)
  film-still.jpg          -> Media Gallery page, documentary section (full width x220px)
  gallery-01.jpg through gallery-06.jpg -> Media Gallery photo grid tiles

Until you add real images, each spot shows a clearly labeled placeholder
so the layout still looks complete and professional.

Tip: keep file sizes reasonable (under ~500KB each) so pages load quickly.
